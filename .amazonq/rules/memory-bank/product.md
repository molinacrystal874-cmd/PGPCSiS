# PGPCSiS - Performance Evaluation System

## Project Purpose
PGPCSiS (Performance Grading and Performance Calculation System in Schools) is a comprehensive web-based evaluation system designed for educational institutions to conduct systematic performance assessments of teaching and non-teaching staff.

## Key Features
- **Multi-Role Authentication**: Secure login system with role-based access (admin, student)
- **Dual Evaluation System**: Separate evaluation workflows for teaching and non-teaching staff
- **Survey Management**: Dynamic survey creation, editing, and management with customizable questions
- **Real-time Analytics**: Dashboard with response tracking, department-wise statistics, and visual charts
- **Ranking System**: Automated calculation and display of staff performance rankings
- **Data Export**: Excel export functionality for evaluation results and staff data
- **Email Integration**: Automated reminder system for pending evaluations
- **Responsive Design**: Mobile-friendly interface with dark/light theme support

## Target Users
- **Educational Administrators**: Manage surveys, view analytics, export reports
- **Students**: Participate in teacher evaluations through structured surveys
- **Academic Institutions**: Schools and colleges requiring systematic staff evaluation

## Use Cases
- Annual teacher performance evaluations by students
- Non-teaching staff assessment and feedback collection
- Performance ranking generation for administrative decisions
- Data-driven insights for institutional improvement
- Compliance with educational evaluation standards

## Value Proposition
Streamlines the traditionally manual process of staff evaluation, providing institutions with reliable data, automated calculations, and comprehensive reporting capabilities for informed decision-making.
